# Personal Portfolio

## Objective
Initial environment setup and toolchain configuration for building an AI-assisted personal portfolio. This serves as Step 1 of the 100hires engineering assessment.

## Tools Installed
*   **IDE:** Cursor (Version: Latest)
*   **Extensions:**
    *   Claude Code
    *   Codex

## Steps Completed
1. Installed Cursor IDE as the primary development environment.
2. Navigated the IDE interface to locate the Extensions marketplace.
3. Installed and authenticated the Claude Code and Codex add-ons.
4. Created a public GitHub repository named `personal-portfolio`.
5. Cloned the repository locally into Cursor.
6. Drafted this README.md to document the setup process.
7. Committed and pushed the initial setup back to GitHub.

## Issues Encountered & Solutions
*   **Issue:** Upon the initial launch of Cursor, the Extensions marketplace wasn't immediately visible in the default UI layout. 
*   **Solution:** I utilized the command palette and the standard VS Code keyboard shortcut (`Cmd + Shift + X` / `Ctrl + Shift + X`) to force-open the extensions panel, allowing me to proceed with the installations without breaking momentum.
